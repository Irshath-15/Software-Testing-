This project focuses on applying manual software testing techniques to validate the functionality, usability, and reliability of a software application. It demonstrates the practical implementation of testing concepts such as test case design, test execution, defect reporting, and regression testing.

---

## Project Objective

The objective of this project is to ensure the quality of a software application by identifying defects early and verifying that the system meets functional and user requirements.

---

## Key Activities

- Requirement analysis  
- Test scenario identification  
- Test case creation and documentation  
- Manual test execution  
- Defect identification and reporting  
- Regression testing after bug fixes
